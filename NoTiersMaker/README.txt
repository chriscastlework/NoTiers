
Custom Logic Projects holds all logic for a project 
Custom Logic Project uses Castles No-tiers Architecture 
Castles No-tiers Architecture automatically handles many basic functions:
Insert 
Bulk Inserts
Update 
Bulk Updates
Delete
Bulk Deletes
View and View Many (Query) 


Core - This folder contains the heart of the architecture and should be modified with extream caution

Services - This holds your services e.g Deals, Accounts, Users, Targets etc.

NOTE: Using No-tiers Maker projects you can automatically make files by connecting to a MSSQL database
